# Tabular Data Lab

10 hands-on tasks — NumPy, Pandas, EDA. Block B of a self-study practicum.

## Tasks

- [11-numpy-basics](./11-numpy-basics/stats.py) — mean, std, covariance, pearson correlation from scratch
- [12-broadcasting-drill](./12-broadcasting-drill) — vectorized normalization, no loops
- [13-image-as-array](./13-image-as-array) — images as NumPy arrays: greyscale, rotation, manual convolution
- [14-pandas-warmup](./14-pandas-warmup) — Titanic dataset, groupby + agg
- [15-pivot-melt](./15-pivot-melt) — long ↔ wide reshaping
- [16-time-series-resample](./16-time-series-resample) — tick data → OHLC candles
- [17-window-functions](./17-window-functions) — rolling, expanding, cumulative — no loops
- [18-eda-airbnb](./18-eda-airbnb) — exploratory analysis on Airbnb listings
- [19-data-quality-report](./19-data-quality-report) — custom `profile(df)` function (no pandas-profiling)
- [20-polars-speedup](./20-polars-speedup) — task 18 rewritten in Polars, benchmarked vs Pandas

## Stack

Python · NumPy · Pandas · Matplotlib · Seaborn · Polars · Jupyter

## Approach

Write the logic by hand first, then compare against the reference implementation.
After each task, log a short entry in `NOTES.md`: what was built, what was learned,
where I got stuck, time spent.
